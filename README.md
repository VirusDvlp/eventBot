Telegram bot in wich certainn users(called "organaizers") can create event with parametres such as name, description, ticket price and date. Standard users can browse these events and buy tickets's. Organizers can switch between user mode and organaizer mode.
Also bot has administrators who can manage organaizers(create them or delete). Admins also have organaizer's permissions.


Bot developed with aiogram framework. Also it used MySQL database

To launch the bot the following environment variables must be set: TOKEN (bot's telegram token), HOST(database's host), DB_USER(databases's user), PASSWORD(user's password), DB_NAME, PORT
